# Blockudoku Dragon

퍼즐앤드래곤식 전투 구조에 블록도쿠 배치 퍼즐을 붙인 웹 프로토타입입니다.

## 실행

```powershell
cd C:\Users\user\Documents\Playground\blockudoku-pad
node server.js
```

브라우저에서 <http://127.0.0.1:5173>을 엽니다.

Admin 페이지는 <http://127.0.0.1:5173/admin.html>입니다.

## 현재 규칙

- 블록 조각은 마우스와 모바일 터치 모두 드래그 앤 드롭으로 배치합니다.
- 행, 열, 3x3 박스를 채우면 칸이 사라지고 콤보 피해가 들어갑니다.
- 블록 색은 팀원 속성처럼 작동합니다.
- 콤보가 없어도 작은 견제 피해가 들어갑니다.
- 적은 3턴마다 반격합니다.
- 팀원 카드를 누르면 속성별 스킬을 사용합니다.

## 도형 DB

- 도형 데이터는 `db/shapes.json`에 저장됩니다.
- Admin 페이지에서 5x5 격자를 눌러 도형을 만들고 저장할 수 있습니다.
- 저장된 도형은 켜기/끄기/삭제/수정이 가능합니다.
- 게임은 시작할 때 켜져 있는 도형만 불러옵니다.
